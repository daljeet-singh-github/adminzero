# Power Automate Naming Convention

Use consistent names so flows are easier to find, support and hand over.

## Format

```text
AZR - [Workflow Area] - [Action]
```

## Referral intake examples

```text
AZR - Referral Intake - Create Tracker Record
AZR - Referral Intake - Save Documents
AZR - Referral Intake - Send Confirmation
AZR - Referral Intake - Missing Info Reminder
AZR - Referral Intake - Notify Admin Team
AZR - Referral Intake - Update Status
```

## Notes

- Prefix AdminZero-built flows with `AZR`.
- Use plain English action names.
- Keep production and test flows clearly labelled.
- Document trigger, owner, connected services and known limitations before handover.
