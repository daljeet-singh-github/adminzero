# AdminZero Company Profile

AdminZero Pty Ltd is a Melbourne-based practice automation consultancy for allied health practices in Australia.

## Summary

AdminZero helps allied health practices automate referral intake, document handling, follow-ups and admin tracking using Microsoft 365 tools they may already have.

The business focuses on practical workflow automation, not custom software or clinical system replacement.

## Audience

AdminZero works with allied health practices and service-based providers, including:

- Physiotherapy practices
- Occupational therapy practices
- Psychology practices
- Rehabilitation providers
- NDIS-adjacent providers
- Case management providers
- Multidisciplinary clinics

## Main Offer

Referral intake automation for allied health practices using Microsoft 365.

## Services

- Free Practice Admin Audit
- Quick Win Automation
- Referral Intake System
- Practice Automation Care Plan
- Email and attachment automation
- Missing information follow-ups
- Admin dashboards and trackers
- Document storage and naming
- Optional AI-assisted document prefill

## Microsoft 365 Tools

AdminZero starts with:

- Microsoft Forms
- Outlook
- SharePoint
- Power Automate
- Microsoft Lists
- Teams

Dataverse, Power BI and AI extraction are used only where suitable.

## Packages

Free Practice Admin Audit: free written review that identifies 3-5 practical ways a practice can reduce manual admin.

Quick Win Automation: from $950, one repetitive admin process automated in 7-10 days.

Referral Intake System: from $3,500, a complete workflow for referral capture, documents, status tracking, missing information and notifications.

Practice Automation Care Plan: from $300/month for monitoring, support and improvements.

## No-Call Audit Process

Website visitor submits the audit form.
AdminZero reviews the current referral or admin workflow.
The practice receives a written summary by email.
No sales call is required.

AdminZero treats AI as an admin assistant, not a decision-maker.
