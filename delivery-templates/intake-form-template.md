# Referral Intake Form Template

## Referrer details

- Referrer name
- Referrer organisation
- Referrer email
- Referrer phone
- Referral source

## Organisation details

- Practice or provider name
- Location
- Primary contact
- Preferred contact method

## Client or claimant details

- Client name
- Date of birth, if required and appropriate
- Contact details
- Claim or reference number, if applicable
- Consent confirmation

Use only the fields genuinely required for the workflow. Avoid collecting sensitive client information unless it is necessary and agreed.

## Referral type

- Service requested
- Priority
- Funding or scheme type
- Preferred clinician or team, if relevant

## Supporting documents

- Referral document
- Reports
- Claim documents
- Other supporting files

## Consent and confirmation

- Confirmation that the referrer is authorised to submit the referral
- Confirmation that information provided is accurate
- Privacy acknowledgement

## Additional notes

- Free text notes for context
- Internal triage notes, if needed
