# Dashboard Layout Template

## Top summary

- Total new referrals
- Pending information
- In progress
- Completed
- Overdue

## Operational views

- Referrals by source
- Referrals by assigned person
- Referrals by status
- Referrals by due date
- Missing information queue

## Optional metrics

- Average processing time
- Number of referrals received this week
- Number of overdue referrals
- Common missing fields

## Design notes

- Keep the first view operational, not decorative.
- Use status filters that match the referral tracker.
- Avoid clinical decision metrics unless specifically scoped and approved.
- Use fake or de-identified data for demos and testing.
