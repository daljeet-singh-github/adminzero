# SharePoint Folder Structure Template

Use a predictable structure so documents are easy to locate and automations can save files consistently.

```text
Referrals /
  Year /
    Month /
      Referral Reference - Client Name /
        Original Referral
        Supporting Documents
        Internal Notes
```

Example:

```text
Referrals / 2026 / April / REF-1024 - Sample Client
```

Recommended naming rules:

- Use a unique referral reference in every folder name.
- Avoid special characters that can cause sync or automation issues.
- Use fake or de-identified data in demos and testing.
- Confirm naming conventions with the client before go-live.
